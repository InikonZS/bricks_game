How to play
To complete level field should be cleared of all colored blocks inside
Player can shot blocks by clicking stacks on the side panels. Shot is not possible when this line is empty
All blocks appeared from side panels save direction, and they will continue moving when cell before got empty. Direction displayed as arrow inside cell
One move can remove many color clusters one by one, displayed as combo message
If block continue to move outside board, it will added to stack as first cell and shift existing cells of stack,
